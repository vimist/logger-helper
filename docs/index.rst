Installation
------------

Installing Logger Helper is super simple, you can either use ``pip``:

.. code:: bash

   pip install logger-helper

or you can install it manually after having cloned or downloaded this
repository:

.. code:: bash

   python3 setup.py install

You're ready to go! Read the :ref:`getting-started` guide for information on
what to do next!
